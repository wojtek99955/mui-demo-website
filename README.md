# Mui demo app

## About

App created with MUI

## Live

https://wojtekk-mui-app-demo.netlify.app/

## Technologies

- reactjs
- formik
- material ui
- yup
- react-context

## Features

- update profile
- add / delete posts
- add post to favorites
- edit posts
- private routes
- user stats: posts, likes, comments number

## Installation

Download or clone the repository

````bash
  npm install
  npm run dev
```
````
